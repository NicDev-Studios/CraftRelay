# Contributing to CraftRelay

Thank you for your interest in contributing to CraftRelay!

## Getting Started

- Fork the repository.
- Create a new branch for your changes.
- Make your changes and test them.
- Commit your changes with a clear commit message.
- Open a Pull Request.

## Guidelines

- Follow the existing code style.
- Keep changes focused and well documented.
- Write clear, maintainable code.
- Add or update tests when applicable.
- Ensure the project builds successfully before submitting a Pull Request.

## Reporting Issues

If you find a bug or have a feature request, please open a GitHub Issue with as much detail as possible.

## Questions

If you have any questions, feel free to open a discussion or contact us.

Thank you for helping improve CraftRelay! ❤️
